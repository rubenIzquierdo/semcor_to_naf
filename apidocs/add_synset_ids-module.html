Module add_synset_ids

source code

This script adds the synset identifiers to some NAF files created with the semcor to NAF converter. In the original NAF files only senses and lexical keys are included. Different types of input is allowed: 1) A folder as generated by conver_corpus.py script 2) A list with paths to NAF files 3) A single file

The synset information is added to the same files, so no new files are created. Run the script with -h option to see the parameters
